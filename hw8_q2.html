Homework #8 Q2

1) Appealing Website: Apple

a) URL: https://www.apple.com

b) Does this page use tables? No, Apple does not use HTML tables for layout. The page uses modern CSS Flexbox and Grid to create a highly responsive design that adapts perfectly to phones and computers.

c) Does this page use CSS? Yes, it uses CSS extensively for page layout, typography, sleek dark/light mode colors, spacing, and smooth scrolling animations.

d) Is this page appealing or unappealing? This page is highly appealing to me.

  • The interface is minimalist and clean, making it very easy to focus on the products.
  • The visual design uses high-quality images and a lot of white space to look premium.
  • The navigation bar is simple and sticky, so you never get lost while scrolling.

2) Unappealing Website: Craigslist

a) URL: https://www.craigslist.org

b) Does this page use tables? Yes, Craigslist still uses basic HTML tables and very rigid column structures to organize its massive lists of links, which is a very outdated design approach.

c) Does this page use CSS? Yes, but very minimally. It uses CSS just for basic text colors and simple spacing, completely ignoring modern layout techniques or visual aesthetics.

d) Is this page appealing or unappealing? For me, this page is unappealing in this comparison.

  • There is way too much text displayed at once, making it look like a messy directory from the 90s.
  • The page feels completely devoid of brand identity, with no engaging images or modern UI elements.
  • Important categories are just giant walls of blue links, which can cause eye strain and makes it overwhelming to browse.

e) If this page is unappealing, what would you do to improve it?

  • Replace the wall of text with clean, card-based layouts using modern CSS Grid.
  • Add high-quality icons or thumbnails next to the main categories to make it visually engaging.
  • Introduce more white space and a modern color palette to make it easier to read and navigate.
